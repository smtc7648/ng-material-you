# NgxMaterialWeb
This library was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.0. This library is the angular implementation of the Google's Latest Material Design, [Material 3](https://m3.material.io/) for web [Material Web](https://m3.material.io/develop/web). Material Web is Googleâ€™s component library for building applications that work in any web framework.
